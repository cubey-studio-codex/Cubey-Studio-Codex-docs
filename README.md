# Cubey Studio

Cubey Studio is an open-source development platform and IDE designed for games,
creative tools and interactive applications.

## Ecosystem

Cubey Studio is composed of multiple independent projects:

- **Cubey Studio** — The main IDE (C# + .NET MAUI)
- **Cubey Web Editor** — Monaco-based code editor (HTML + JavaScript)
- **Cubey Engine** — Native C++ engine (build, runtime, tools)
- **Cubey Language Server** — Language Server Protocol implementation
- **Cubey Plugins** — Official and community extensions

## Technology Stack

- C# / .NET MAUI
- WebView
- HTML, CSS, JavaScript
- Monaco Editor
- C++ (native core)

## Design Goals

- Modular architecture
- Extensible plugin system
- Cross-platform support
- Professional-grade tooling
